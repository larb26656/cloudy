# Memory Content

วันนี้ผมคิดถึงการทำฟีเจอร์ใหม่บนหน้าเว็บที่ให้ผู้ใช้สามารถลากวาง widget ได้เอง โดยแต่ละ widget จะมีสถานะเป็น dynamic data และสามารถเชื่อมต่อกับ API ต่าง ๆ ได้

## Details

- **Widget Types:** chart, table, text, image
- **Interactivity:** draggable, resizable
- **Data Sources:** REST API, GraphQL
- **Future Enhancements:** real-time updates, user presets

## Notes

- ต้องออกแบบให้ state management รองรับ widget หลายตัว
- พิจารณาใช้ localStorage เก็บ layout ชั่วคราว
- ทดสอบ performance กับ dataset ขนาดใหญ่
